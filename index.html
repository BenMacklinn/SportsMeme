How It Works

Prototype logic modeled after Techmeme-style curation.

  1. Ingest coverage from a curated set of sports publishers.
  2. Cluster reports that describe the same sports event.
  3. Rank each cluster by recency, source depth, and buzz.
  4. Render a front page and a reverse-chronological river.
